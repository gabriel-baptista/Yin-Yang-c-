Projeto de gerenciamento de clinicas nutricionistas "Yin Yang"
autores: Gabriel Oliveira, João Santos, Rodrigo Florentino, Alberto Sabino, Igor Eduardo


como abrir: 
1) entrar na pasta do seu usuario do sistema operacional

2) entrar na pasta source/repos e colar a pasta "Yin Yang"

3) subir um servidor local (caso precise, mude as portas de conexao, usuario e senha dentro do código do sistema)

4) criar um banco sql com o código que está dentro do arquivo "Yin Yang/banco"

5) executar o sistema

6) dar 10 pra gente

7) ser feliz ;)
